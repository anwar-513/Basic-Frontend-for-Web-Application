Selectors

Patterns to select the elements you want to style

Selectors are patterns which is used to select one or more elements from the HTML document. It gives the ability to target the specific elements have to style. There are different types of selectors available in CSS. The most common ones are:

  • Universal selector, syntax: *
  • Type selector, syntax: element-name
  • Class selector, syntax: .class-name
  • ID selector, syntax: #id-name
  • Group selector, syntax: element1, element2
  • Descendant selector, syntax: ancestor descendant
  • Child selector, syntax: parent > child
  • Adjacent sibling selector, syntax: element1 + element2
  • General sibling selector, syntax: element1 ~ element2

Every different selector has its own different syntax.

Types of Selectors

Universal Selector

The universal selector is represented by an asterisk (*) and it selects all elements in the document. It is often used to apply a common style to all elements or to reset default styles.

Type Selector

The type selector targets elements based on their HTML tag name. For example, using the type selector "p" will select all paragraph elements in the document.

Class Selector

The class selector targets elements based on their class attribute. It is denoted by a period (.) followed by the class name. For example, ".example" will select all elements with the class "example".

ID Selector

The ID selector targets a single element based on its unique ID attribute. It is denoted by a hash (#) followed by the ID name. For example, "#unique" will select the element with the ID "unique".
