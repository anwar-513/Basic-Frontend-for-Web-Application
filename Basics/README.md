# HTML Basics

This folder contains lessons on HTML fundamentals including:
- Tags and tag definitions
- Tag types
- Input tags
- Attributes
- Boolean attributes

Files in this section follow The Odin Project's "Complete Intro to Web Dev v3" curriculum.

## Files in this Directory

### Lesson Files

1. **1.01_tags_definition.html** - Introduction to HTML tag definitions
2. **1.02.01_tags_types.html** - Different types of HTML tags
3. **1.02.02_input_tags.html** - HTML input tags and their usage
4. **1.02.03_tags_types.html** - Additional tag types and classifications
5. **1.03.01_Attributes_in_HTML.html** - HTML attributes explained
6. **1.03.02_Boolean_Attributes.html** - Boolean attributes in HTML
7. **1.04_first_webpage.html** - Creating your first webpage
8. **1.05.01_Organizing_code.html** - Code organization basics
9. **1.05.02_Organizing_code.html** - Advanced code organization techniques
