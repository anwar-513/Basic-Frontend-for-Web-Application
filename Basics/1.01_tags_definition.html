Tags

It is my first ever code written in HTML Tags are building blocks, describing what is inside it. Every tag has an opening and closing tag. The opening tag is written with angle brackets as and the closing tag is written with angle brackets, in addtion a slash as .

Here the content of this page is written in the title tag, It is also called heading tag. title/heading tags are used to define the heading of a page. It is bigger and bolder than normal text. Heading tags have six levels, from h1 to h6.

Heading tags are important for blind and people to understand what is important on the page. They are also important for search engines like google and bing to understand the structure of the page. It is nessary for coders to understand which level/king of heading tag is to be used for which content.

It is important to keep in mind the order of tags. Opening and closing tags should be in the correct order. For example, if you open a tag and then open another tag, you should close the second tag before closing the first tag. This is called nesting of tags.
