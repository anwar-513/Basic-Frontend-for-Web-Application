ATTRIBUTES

Attributes allow us to modify the behavior of HTML tags. Simple, attributes give extra information to a tag. They provide additional information about an element and are always included in the opening tag. Attributes consist of a name and a value which are separated by an equal sign.

IDs and classes are two of the most commonly used attributes in HTML. while both are used to target the element for styling and behavior they serve in very different purposes.

Classes can be used and reused on as many element you want while Id is used for unique and only be used for one element. Moveover, an id can link you to the specific portion of a page.
