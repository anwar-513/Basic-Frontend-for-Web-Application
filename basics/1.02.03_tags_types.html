Further Types of Tags

1. The "Textarea" tag is used to create a multi-line text input field. It is same as input tag but it is used for longer text input. it is a paired tag.





2. The "select" tag is used to create a drop-down list. It is a paired tag. It is used to create a list of options for the user to choose from. The "option" tag is used to define the options in the drop-down list.



3. The "form" tag is used to create a form. It is a paired tag. just like button it does not have any meaning by itself but it is defined by what is inside it. It is used to group the input elements together. In the form tag there should be something like a combination of input, textarea, and select tags.









4. If you have to display some data in table form,The "table" tag is used for it. It is a paired tag. Just like making tables in MS word or excel, the table tag is used in HTML for display data in rows and colums. The "tr" subtag of table tag is used for the rows of a table, while the "td" tag represent one cell of the table. The "th" tag is used for the header of the table.
