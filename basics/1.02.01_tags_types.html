Types of Tags

There are several types of HTML tags: In HTML5 the number of tags has increased to 145. In addition, the numbers of tags depends on how you are using them.
Moreover, tags are classified into two types:
1. By structure
2. By function

By structure:
Tags by structure fall into two categories:
1. paired tags
2. Self-closing tags

Paired tags:
paired tags are the tags with opening

< name >

and closing tags

< /name >

and Paired tags wrap content and define structure.

Self-closing tags:
Self-closing tags are single and standalone tags. Self-closing tags handle tasks like line breaks

< br >

and images

< img >

By Functions:
HTML tags are grouped into functional categories including: document structure, sections and grouping, headings and content, text-level semantics, embedded content, forms, tables, interactive elements, and metadata.
